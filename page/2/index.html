无意间的一次玩弄grunt，发现了hexo，就花了点时间玩了玩,建了个博客。

安装过程

如果本机安装了node 忽略下面一段

安装git 和 node

这个就不介绍了，分别直接官网 git node 下载安装就好了，建议全局安装，不要逼格太高瞎折腾。其实也没啥关系，猿猿都爱折腾。

错误解决

// 各种npm安装的插件对其操作命令过程中，如果系统提示没有这个命令，基本上不是需要update,就是没有全局安装，需要在系统路径下注册，即弄个环境变量。比如安装hexo$ npm install -g hexo发现$ hexo -V不好使，那就找到你安装hexo的根目录，copy路径，右键我的电脑 > 属性 > 高级系统设置 > 环境变量 > 点开 系统变量Path 末尾添加类似路径C:\Program Files\nodejs\node_global。你就会发现你安装的插件都好用啦，比如grunt \ hexo
